project overview:

This project is a comprehensive portfolio demonstrating core digital competencies, ranging from professional communication and online presence to cybersecurity awareness. It consists of five distinct modules/tasks designed to showcase practical application of digital tools and professional etiquette.

Module Summaries:

Module 1:

In module one we were taught to varies digital tools like canva , prezi and googledocs to create presentation which is part of a students life and these tools makes it easier to create presnetation which we can use for seminars,projects,etc. The task given from module 1 is to create a infograph using prezi or canva that covers at least 3 of the following topics: what digital literacy is, useful digital tools for students, safe internet practices, professional online presence, email etiquette. Here i used prezi to create a infograph on the topics:what digital literacy is, useful digital tools for students, safe internet practices. Module 1 mainly focuses on visual communication and presentation design.

module 2:

module 2 mainly focuses on building a digital portfolio in linkedin, github, kaggle, stack overflow, research gate. These platforms acts as a intermediate between students and many organizations which helps to get many opportunity. The task in module 2 is to create any three accounts from platforms listed above. Focus: Digital identity and professional branding.

Module 3:

Module 3 introduces us to varies coding platforms such as HackerRank, CodeChef, HackerEarth. Coding is about practice and these platforms will be helpful to learn/practice many languages. The task in module 3 is to create an account in any one of the platform and complete the biginner level challenge and to create a google form which consists 5 questionsfor awareness in digital literacy. Focus: Platform engagement and digital tool proficiency.

Module 4:

In module 4 we were taught to communicate with certain rules which helps to professional email with no mistake and etiquette guide. The task here is to draft 2 proffesional emails and a checklist for social media do's and dont's. Focus: Professional correspondence and communication standards.

Module 5:

This module focuses on the critical analysis of digital threats and the implementation of security best practices to protect the personal and professional data. The task here is to create a case study for any one of the following topics: Phishing, UPI/Online payments fraud, Identity Theft, Cyberbullying, ransomware, and a prevention checklist. Focus: Digital security and risk mitigation.
